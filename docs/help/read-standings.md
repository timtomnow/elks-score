The standings card shows the full league table by division, so you can see exactly where the Elks rank and who they're chasing.

## Steps

1. Scroll to the **CFL Standings** card.
2. Read the two tables: **West Division** and **East Division**, each sorted by points.
3. Across the columns: **Team**, **GP** (games played), **W**, **L**, **T**, and **Pts**. CFL scoring is win = 2, tie = 1, loss = 0.
4. Find the highlighted row — that's the Elks.

## What you'll see

Both divisions ranked top to bottom, with the Elks called out so you can spot their position at a glance.
