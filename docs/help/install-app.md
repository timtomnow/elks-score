Elks Score is a Progressive Web App, so you can install it straight from your browser — no app store needed. Once installed it opens full-screen and keeps the app shell available offline.

## Before you start

- Open the app in your phone's browser (Chrome on Android, Safari on iPhone).

## Steps

1. **On Android (Chrome):** tap the **⋮** menu, then **Add to Home screen** (or accept the **Install app** prompt if it appears).
2. **On iPhone (Safari):** tap the **Share** button, then **Add to Home Screen**.
3. Confirm the name and tap **Add**.

## What you'll see

An Elks Score icon on your home screen that launches the app full-screen. Live score data always loads fresh from the network; only the app shell is cached for offline use.
