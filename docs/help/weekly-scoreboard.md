The weekly scoreboard lists every game in the current round, not just the Elks', so you can keep an eye on the whole league at once.

## Steps

1. Scroll to the week card — it's titled with the current round (for example **Week 5**).
2. Read each row: the two teams with the score, or an **@** for games that haven't started.
3. Check the status on each row — the live period and clock, **Final**, **Canceled**, or the kickoff time for upcoming games.
4. Tap any game row (look for the **›**) to open its official CFL.ca Game Tracker page.

## What you'll see

The full slate for the week. The Elks' game is highlighted when they're playing, and live games show their clock so you can tell at a glance what's still in progress.
